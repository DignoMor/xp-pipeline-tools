# xp-pipeline-tools

Pip-installable multi-entrypoint tools distribution.

## Install

```bash
pip install -e .
# or with dev extras:
pip install -e ".[dev]"
```

## Console scripts

After install, `buddingScripts` is available on `PATH`. You can also run
`python -m buddingScripts`. The package is importable as
`import buddingScripts`.
